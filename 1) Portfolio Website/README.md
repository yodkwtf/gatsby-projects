# Portfolio Website

A portfolio website created with gatsby.

## Preview

[Portfolio Website - Live](https://portfoliowebsite-gatsby-dk.netlify.app/)

## Stack

- ReactJs + Gatsby.js
- React Hooks
- CSS
- GraphQL
- Starpi as CMS

## How to Run?

To run this project on your system, run

```bash
  npm install
  npm start
```

To create a build of this project, run

```bash
  gatsby build
```
